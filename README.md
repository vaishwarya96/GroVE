# GroVE

## Dataset
The code supports the MS-COCO, Flickr30k, CUB-200-2011 and Oxford Flowers 102.

COCO: Download the 2014 data and setup the directory in the following way

```bash
coco
|-images/
|--train2014 
|--val2014 
|-captions_train2014.json 
|-captions_val2014.json
```
